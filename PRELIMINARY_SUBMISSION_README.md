# 时刻 (Shike) - 提交说明

> 一句话整理提醒，不错过重要时间。

## 产品定位

**时刻** 是一款轻量级提醒整理工具，用户输入一句自然语言描述（如"明天下午3点交报告"），系统自动识别时间、事项和优先级，生成可编辑草稿，确认后保存到本地。

- **用户可见产品名**：时刻
- **项目代号**：Memorial day
- **版本**：Final Functional MVP
- **公网体验**：https://memorialdaylan.netlify.app/

## 核心功能闭环

1. **输入**：首页输入框直接可用，支持示例 chip 一键填入
2. **解析**：本地规则识别时间词（明天/下周/月底/上午/下午/X月X日等）
3. **草稿**：解析结果展示为可编辑草稿卡片，用户可修改后再保存
4. **保存**：确认后写入 localStorage（key: `shike_reminders_v1`）
5. **展示**：今天 / 日历 / 全部三个视图真实渲染数据
6. **详情**：点击提醒卡片打开抽屉查看完整信息
7. **完成**：勾选 checkbox 标记完成，已完成项加删除线
8. **删除**：支持删除提醒
9. **持久化**：刷新页面数据不丢失

## 诚实声明

- 当前为本地规则识别，**未接入真实大模型**
- 所有数据保存在用户浏览器 localStorage，**不上传服务器，不需要账号**
- TXT/PDF/DOCX 文档解析后续版本支持
- Memorial day 是项目代号，用户可见产品名为"时刻"

## 技术栈

- 纯 HTML + CSS + JavaScript 单文件应用，无外部依赖
- 移动端优先响应式设计
- PWA 支持（manifest + Service Worker）
- Service Worker 使用 network-first 策略确保 HTML 及时更新
- GitHub + Netlify 持续部署，不使用 Netlify Drop
