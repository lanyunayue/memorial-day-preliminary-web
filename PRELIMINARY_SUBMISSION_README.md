# 时刻 v0.6.0

**你的贴心记事助手**

## 一句话介绍

时刻是一款轻量级记事工具。输入一句话，自动整理标题、日期、时间，保存为本地记录。有日期的事项进入日历并可设置系统提醒。支持中文数字时间、近时间分钟级倒计时、精美大卡片展示。

## 在线体验地址

为保证不同网络环境下的访问稳定性，项目提供主站与备用镜像。如遇某个平台访问异常，可切换至备用地址。

- **主站（GitHub Pages）**：https://lanyunayue.github.io/memorial-day-preliminary-web/
- **海外备用（Cloudflare Workers）**：https://memorial-day-preliminary-web.308138249.workers.dev/
- **历史镜像（Netlify）**：https://memorialdaylan.netlify.app/

## 核心特性

1. **一句话记录**：在首页输入框输入任意内容，点击保存即可。支持缺日期、缺时间、缺地点，只要有文字就能保存。
2. **智能时间解析**：本地规则识别中文数字时间（九点、三点半、九点钟）、相对时间（待会、等一下、一会儿、稍后、马上、半小时后、N小时后）、时间段默认今天（下午三点→今天15:00）。
3. **标题清洗**：自动去除时间词、介词，保留核心动作，不会出现"吃饭种"这种错误。
4. **批量导入**：粘贴一段多行文字，每行自动拆分为一条草稿，逐条或全部保存。
5. **日历标记**：有日期的记录自动在月历和本周日历条上显示圆点标记。
6. **系统通知**：有日期时间的记录可设置当天/提前一天/提前三天提醒，使用浏览器 Notification API。
7. **本地存储**：数据保存在浏览器 localStorage，刷新不丢，可导出 JSON 备份。
8. **通用大卡片**：所有类型（提醒/纪念/习惯/备忘）均可切换为大卡片展示，支持自定义背景图、6套预置渐变、置顶。
9. **实时倒计时**：近时间提醒显示分钟/小时级倒计时（如"还有29分钟"、"还有18小时24分钟"），每分钟自动刷新。
10. **完整闭环**：输入→保存→展示→查看详情→编辑→隐藏→删除。
11. **5套主题**：雾蓝 / 暖米 / 墨黑 / 松青 / 云母，低饱和度高级配色。
12. **语音输入**：Chrome/Edge/Safari 支持语音识别；微信内置浏览器自动隐藏语音按钮，不显示坏按钮。
13. **开场动画**：1.8秒高级品牌动画，可跳过，第二次访问不打扰。

## 诚实说明

- 本地规则整理，未接入真实大模型
- 系统通知使用浏览器 Notification API，需要用户授权
- 数据保存在当前浏览器，清除浏览器数据会丢失记录
- Web 与 HarmonyOS 当前不实时同步
- 微信内置浏览器不支持语音输入（已自动隐藏按钮）
- PDF/DOCX 解析后续支持

## 技术信息

- 版本：v0.6.0
- 纯前端单文件应用（HTML+CSS+JS），无框架依赖
- PWA 支持，可添加到桌面
- Service Worker：network-first 策略确保始终加载最新版本（cache v16）
- 5套主题可选
- 响应式设计，移动端优先
- 60条 NLP 解析测试用例全部通过
