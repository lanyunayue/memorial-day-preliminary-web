# 时刻 v0.7.0

**你的贴心记事助手**

## 一句话介绍

时刻是一款轻量级记事工具。输入一句话，自动整理标题、日期、时间，保存为本地记录。有日期的事项进入日历并可设置系统提醒。支持中文数字时间、近时间分钟级倒计时、精美大卡片展示、多语言、节假日、天气。

## 在线体验地址

为保证不同网络环境下的访问稳定性，项目提供主站与备用镜像。如遇某个平台访问异常，可切换至备用地址。

## 核心特性

1. **一句话记录**：在首页输入框输入任意内容，点击保存即可。支持缺日期、缺时间、缺地点，只要有文字就能保存。
2. **首页极简设计**：无记录时输入框居中、实时日期时间，首条保存后输入框平滑过渡到顶部（300-500ms）。
3. **智能时间解析**：本地规则识别中文数字时间（九点、三点半、九点钟）、相对时间（待会/等一下/一会儿→+30分钟、稍后→+1小时、马上→+10分钟、半小时后、N小时后）、时间段默认今天（下午三点→今天15:00，晚上九点→今天21:00，凌晨一点→今天01:00）。
4. **标题清洗**：自动去除时间词、介词、语气词，保留核心动作，"明天睡觉八点钟"→"睡觉"、"下午三点交实验报告"→"交实验报告"，不会出现"睡觉钟""吃饭种"。
5. **批量导入**：粘贴一段多行文字，每行自动拆分为一条草稿，逐条或全部保存。
6. **日历标记与节假日**：有日期的记录自动在月历上显示圆点标记；内置2026年常见节假日高亮；支持新历/农历切换。
7. **系统通知**：有日期时间的记录可设置当天/提前一天/提前三天提醒，使用浏览器 Notification API。
8. **本地存储**：数据保存在浏览器 localStorage，刷新不丢，可导出 JSON 备份；所有新增字段自动迁移，旧数据向前兼容。
9. **通用大卡片**：所有类型（提醒/纪念/习惯/备忘）均可切换为大卡片展示，支持自定义背景图、8套预置渐变、主题色、置顶；首页最多显示3张置顶大卡片。
10. **实时倒计时**：支持分钟/小时/天级倒计时（"还有29分钟"、"还有18小时24分钟"、"还有3天"），每分钟自动刷新；已过事项显示"已过X分钟/小时/天"。
11. **完整闭环**：输入→保存→展示→查看详情→编辑→隐藏→删除。
12. **8套主题**：米白、黑白、咖啡、雾蓝、玫瑰、森林、夜间、樱花，低饱和度高级配色。
13. **4种语言**：简体中文、繁體中文、English、日本語，切换即时生效。
14. **个性化**：可设置用户名、显示使用天数、首页天气（基于 Open-Meteo，30分钟缓存）。
15. **语音输入**：Chrome/Edge/Safari 支持语音识别；微信内置浏览器自动隐藏语音按钮，不显示坏按钮。
16. **开场动画**：1.8秒高级品牌动画，可跳过，第二次访问不强制展示。

## 诚实说明

- 本地规则整理，未接入真实大模型
- 系统通知使用浏览器 Notification API，需要用户授权
- 数据保存在当前浏览器，清除浏览器数据会丢失记录
- 农历当前优先支持2026年常用显示
- 天气功能依赖 Open-Meteo 公开接口，定位失败/接口失败有降级提示
- 微信内置浏览器不支持语音输入（已自动隐藏按钮）
- HarmonyOS 端核心功能已同步（首页UX/NLP/大卡片/多语言/农历/天气开关）

## 技术信息

- 版本：v0.7.0

- 100条 NLP 解析测试用例全部通过
- 定时器在页面隐藏时降频，避免内存泄漏
