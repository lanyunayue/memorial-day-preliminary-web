# 时刻 v0.5.1

**你的贴心记事助手**

## 一句话介绍

时刻是一款轻量级记事工具。输入一句话，自动整理标题、日期、时间，保存为本地记录。有日期的事项进入日历并可设置系统提醒。

## 在线体验地址

为保证展示稳定性，项目提供多个静态镜像地址。如遇某个平台访问异常，可切换至备用地址。

- **主站（GitHub Pages）**：https://lanyunayue.github.io/memorial-day-preliminary-web/
- **Cloudflare Pages**：待连接（配置文件已就绪，在 Cloudflare Dashboard 连接 GitHub 仓库后即可部署，预计地址为 `https://shike-memorial.pages.dev`）
- **历史镜像（Netlify）**：https://memorialdaylan.netlify.app/

## 核心特性

1. **一句话记录**：在首页输入框输入任意内容，点击保存即可。支持缺日期、缺时间、缺地点，只要有文字就能保存。
2. **自动整理**：本地规则识别"明天"、"下周六"、"晚上"、"睡前"、"3点"等时间表达，自动归类为提醒/纪念/习惯/备忘。
3. **批量导入**：粘贴一段多行文字，每行自动拆分为一条草稿，逐条或全部保存。
4. **日历标记**：有日期的记录自动在月历和本周日历条上显示圆点标记。
5. **系统通知**：有日期时间的记录可设置当天/提前一天/提前三天提醒，使用浏览器 Notification API。
6. **本地存储**：数据保存在浏览器 localStorage，刷新不丢，可导出 JSON 备份。
7. **记录大卡片**：每条记录显示类型、标题、日期时间、倒计时/已记录天数、提醒状态。
8. **完整闭环**：输入→保存→展示→查看详情→编辑→隐藏→删除。
9. **5套主题**：雾蓝 / 暖米 / 墨黑 / 松青 / 云母，低饱和度高级配色。
10. **语音输入**：支持语音识别（Chrome/Edge 推荐），内置状态机确保稳定可用。

## 诚实说明

- 本地规则整理，未接入真实大模型
- 系统通知使用浏览器 Notification API，需要用户授权
- 数据保存在当前浏览器，清除浏览器数据会丢失记录
- Web 与 HarmonyOS 当前不实时同步
- PDF/DOCX 解析后续支持

## 技术信息

- 版本：v0.5.1
- 纯前端单文件应用（HTML+CSS+JS），无框架依赖
- PWA 支持，可添加到桌面
- Service Worker：network-first 策略确保始终加载最新版本
- 5套主题可选
- 响应式设计，移动端优先
