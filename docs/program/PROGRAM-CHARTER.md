# PROGRAM CHARTER

## Mission

把「时刻 Shike」从可演示的本地优先时间记录工具，打磨成可靠、可恢复、可解释、可长期信任的个人时间智能系统。

## Alpha2 Scope

- 只处理投资级质量门禁与真实证据。
- 合并候选只允许 CI、a11y、格式、测试真实性和必要 P0/P1 修复。
- 不发布产品版本，不更新版本号，不更新缓存名，除非真实浏览器与发布门槛全部满足。

## Decision Rights

- P0/P1 修复优先于路线图扩张。
- 真实浏览器证据缺失时，发布默认停止。
- 云能力缺失时，不允许产品文案暗示跨设备可靠同步。

## Non-Goals

- 不提前创建 alpha3。
- 不新增云同步。
- 不新增支付。
- 不新增团队协作。
- 不新增平台端。
