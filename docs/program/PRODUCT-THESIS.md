# PRODUCT THESIS

## Thesis

Shike 的核心价值不是另一个清单或日历，而是把自然语言中的时间承诺快速变成可回看、可提醒、可恢复的个人时间记忆。

## Differentiation Hypotheses

1. 用户愿意用一句话记录未来事件、备忘、习惯和承诺。
2. 用户需要知道数据在哪里，以及提醒什么时候可靠、什么时候不可靠。
3. 长期价值来自每日/每周回顾、未完成承诺和未来七天视图。
4. 本地优先是信任基础，但换设备问题必须被诚实说明或解决。

## Current Evidence Gap

- 没有真实用户激活、留存或付费数据。
- 没有真实浏览器端到端证据覆盖核心路径。
- 没有云账号与跨设备同步基础。

## Current Product Stage

MVP/Alpha 之间。可以继续投入验证与可靠性，不能按可发布商业产品对外声明。
