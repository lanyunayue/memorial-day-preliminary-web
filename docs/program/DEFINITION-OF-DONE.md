# DEFINITION OF DONE

## Code Change Done

- Diff 审计完成。
- 无不相关产品改动。
- `git diff --check` 通过。
- lint、format、unit、legacy、parser、a11y、security、test integrity 通过。
- 浏览器相关项明确 PASS 或 SKIPPED。
- SKIPPED 不计入 PASS。

## Release Done

- 真实浏览器 7 视口验证完成。
- Console/network error 已检查。
- 离线启动和 SW 更新演练完成。
- `APP_VERSION` 与 `CACHE_NAME` 按发布策略更新。
- 部署地址验证完成。

## Documentation Done

- 报告只声明真实完成项。
- 规划项必须标为计划或 backlog。
- 未完成证据必须列为风险。
