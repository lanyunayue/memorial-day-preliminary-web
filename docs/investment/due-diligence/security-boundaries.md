# Security Boundaries

All Chronos intelligence is local. No remote model, account, payment, or cloud synchronization was added. Operation errors and correction text redact common secret forms. Adaptation rules containing secret material or sensitive personality labels are rejected.

The parser gate uses canonical UTF-8/LF SHA-256: `efbff968efd518e26970bac24ad35396df8482a32ba56011c6670167d58c4b58`. The Windows working-tree byte hash is `d6298d52d56beddfc407b329569fe81f179fcf50652425ed29dda6fa6eb6be32` and is diagnostic only.

Browser origin compromise, malicious extensions, and a compromised operating system remain outside this local application boundary.
