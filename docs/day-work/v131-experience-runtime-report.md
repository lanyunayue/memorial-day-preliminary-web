# v1.3.1 体验稳定性修复 - 运行时报告

## 静态验证结果

### 首页滚动修复
- history.scrollRestoration = 'manual' 已设置
- init 流程包含多处 scrollTo(0,0) 保障
- 开场动画关闭后 scrollTo 顶部

- 空状态 topbar 使用 display:none 不占文档流

- html/body 无 padding，避免重复 safe-area
- @media (min-width:1024px) 媒体块完整闭合

### 精灵意图识别
- 40+ 创建动词支持
- 日期时间关键词完整保留
- 填充词语气词剥离
- 问句/闲聊/快捷指令不误判
- XSS 防护（script 标签不执行）
