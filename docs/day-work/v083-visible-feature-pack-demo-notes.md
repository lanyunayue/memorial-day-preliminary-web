# Shike v0.8.3 Visible Feature Pack Demo Notes

## 1. 时间旅程线解决什么问题

v0.8.2 已经能记录、导出和保护数据，但“全部页”仍然偏列表视角。时间旅程线把记录按今天、明天、本周、未来、无日期分组，让评委一眼看到用户的时间流，而不是只看到零散条目。

## 2. PNG 纪念卡片解决什么问题

纪念日是“时刻”最有情绪价值的部分。PNG 纪念卡片让妈妈生日、周年纪念这类大卡片可以本地生成图片，方便保存和展示。它只使用浏览器 canvas，不上传用户数据。

## 3. 今日概览解决什么问题

用户打开首页时，需要立刻知道最近要关注什么。今日概览用很小的空间展示今天记录数、最近安排和最近纪念日，不遮挡主输入框，也不把首页做复杂。

## 4. 如何服务“一句话时间整理入口”

这三个功能都围绕“一句话输入”之后的结果展开：

- 输入一句话后，首页今日概览马上提示最近安排。
- 全部页时间旅程线把记录放回时间语境。
- 纪念日记录可以生成 PNG 卡片，增强情绪价值和复赛展示记忆点。

## 5. 如何和 v0.8.2 底座组合

v0.8.2 提供了 `.ics`、数据安全中心、导入预览、PWA 提醒说明和时刻精灵。v0.8.3 没有替换这些能力，而是在其上增加可见体验层：

- `.ics` 继续负责进入系统日历。
- JSON 备份继续负责数据保护。
- 导入预览继续负责安全迁移。
- 今日概览和时间旅程线负责打开后的理解效率。
- PNG 纪念卡片负责展示与保存。

## 6. 复赛演示顺序建议

1. 首页输入一句话，例如“明天下午三点开会”。
2. 展示今日概览出现最近安排。
3. 点击“体验示例”生成 5 条记录。
4. 进入全部页，展示时间旅程线和原全部列表仍然存在。
5. 搜索“妈妈”，打开妈妈生日纪念卡片。
6. 点击“保存卡片”，说明 PNG 在本地生成，不上传数据。
7. 进入我的页，展示 `.ics` 导出和 JSON 备份。
8. 说明 v0.8.3 是在 v0.8.2 底座上增加可见价值，没有接云、登录或外部 API。
