# 时刻 v0.7.4 初赛发帖文案

线上地址：https://lanyunayue.github.io/memorial-day-preliminary-web/

时刻是一款面向生活场景的轻量记事与倒计时 PWA。它支持用一句自然语言记录提醒、纪念日、习惯和备忘，并在首页、全部记录和日历中自动联动展示。

为方便评委快速体验，v0.7.4 增加“体验示例”入口，可一键生成 5 条高质量示例记录，展示提醒、纪念、习惯、备忘、日历联动和搜索筛选效果。

当前版本：v0.7.4

Service Worker cache：shike-v074-v25

推荐体验路径：

1. 打开正式地址。
2. 点击首页输入区下方的“体验示例”。
3. 查看首页大卡片、下一条提醒和最近记录。
4. 进入“全部”，测试搜索“妈妈”和类型筛选。
5. 进入“日历”，查看日期 dot 和当天记录。

未来展望：后续可以增加纪念卡片导出图片，让用户生成适合分享的纪念海报。
