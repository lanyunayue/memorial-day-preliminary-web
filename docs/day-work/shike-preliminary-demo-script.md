# 时刻 v0.7.4 初赛演示脚本

版本：v0.7.4

线上地址：https://lanyunayue.github.io/memorial-day-preliminary-web/

## 60 秒演示路径

1. 打开首页，说明“时刻”支持用一句话记住生活里的重要时间。
2. 点击“体验示例”，一键生成 5 条记录。
3. 在首页展示妈妈生日大卡片、开会提醒、习惯记录和本周日期联动。
4. 切到“全部”，搜索“妈妈”，展示搜索结果。
5. 在“全部”依次筛选纪念、提醒、习惯、备忘，展示不同记录类型。
6. 切到“日历”，展示日期 dot 和当天记录。
7. 说明数据默认保存在当前浏览器，支持本地备份和恢复。
