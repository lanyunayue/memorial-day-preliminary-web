# v2.0.0-rc1 检索来源治理报告

## Provider列表

### 1. Open-Meteo (天气)
- API URL: https://api.open-meteo.com/v1/forecast
- 是否需要key: 否
- CORS: 允许
- 限流: 免费层10000次/天
- Attribution: Open-Meteo (CC BY 4.0)
- 使用条款: 非商业免费使用
- 商业使用风险: **需重新确认授权**
- 超时: 10秒
- 缓存时间: 30分钟
- 降级策略: 天气卡片隐藏，不影响其他功能
- 最后验证: 2026-07-11

### 2. 内置公开来源 (关注中心)
- 数据来源: 内置静态数据
- 是否需要key: 否
- CORS: 不适用
- 限流: 不适用
- Attribution: 内置公开信息示例
- 降级策略: 显示空状态
- 最后验证: 2026-07-11

### 3. 浏览器内置AI (可选)
- 默认状态: 关闭
- 不可用时: 回退到规则总结
- 数据边界: 不上传用户记录
- 隐私: 所有处理在本地
- 最后验证: 2026-07-11

### 4. 自定义RSS
- 数据来源: 用户提供的RSS URL
- CORS: 受目标服务器限制
- 降级策略: 显示CORS错误提示
- 安全: DOMParser解析，不执行脚本
- 最后验证: 2026-07-11

## 风险声明

1. Open-Meteo免费端授权不适合作为未来商业化时未经复核的默认来源
2. 中文浏览器内置AI支持有限，当前会回退到规则总结
3. 纯前端检索受CORS、限流和公开接口稳定性影响
