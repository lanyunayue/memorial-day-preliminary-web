# 时刻隐私政策草案

时刻·个人事件助手在用户明确授权后，从用户选择的应用通知中本地识别快递状态。我们不读取未获白名单允许的来源，不保存原始通知正文，不自动操作其他应用，不读取短信、聊天数据库、剪贴板历史、位置或麦克风后台数据。

结构化快递数据保存在设备本地。取件码使用 Android Keystore 保护的加密密钥保存，默认遮挡。应用当前不申请网络权限，不上传通知或快递数据，不集成广告、用户画像或第三方行为分析 SDK。

用户可以撤销通知访问、修改来源白名单、删除单条记录、清除全部数据或导出遮挡数据。权限被拒绝或撤销后，手动输入和“分享到时刻”仍可用。

本文件是 Alpha 工程草案，正式发布前必须经过适用司法辖区的法律审阅并补充运营主体与联系方式。
