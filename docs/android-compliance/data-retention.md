# 数据保留

| 数据 | 默认保留 | 删除方式 |
| --- | --- | --- |
| 原始通知标题/正文 | 仅处理期间驻留内存 | 解析后立即丢弃 |
| Parcel 与时间线 | 直到用户删除 | 单条删除或清除全部 |
| 加密取件码 | 随 Parcel 保留 | 删除 Parcel 或清除全部 |
| 待确认草稿 | 直到确认、忽略或清除 | 草稿操作或清除全部 |
| Operation Journal | 本机诊断使用 | 当前 Alpha 随应用数据清除 |
| 白名单与模式 | 直到用户修改 | 设置或撤销并清除 |

应用禁用云备份和设备迁移中的数据库/偏好备份，避免敏感数据脱离本机保护边界。
