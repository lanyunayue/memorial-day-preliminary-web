# 知情同意书

## 真人试用研究：过载恢复机制产品验证

> **HUMAN OVERLOAD PILOT REQUIRED**
>
> 本知情同意书适用于"过载恢复机制"真人试用研究。请在签署前仔细阅读全部内容。如有疑问，可随时联系研究者。

---

## 一、研究目的说明

本研究旨在验证一款个人事件助手应用中"过载恢复"功能的产品假设。具体目标是了解：

- 用户是否能够快速理解签到流程
- 用户是否会主动查看负荷解释和降载选项
- 用户对系统建议的接受、修改或拒绝比例
- 用户是否认为建议"不说教"且"有帮助"
- 用户是否愿意在试用期间持续使用

研究结果将用于改进产品设计，使过载恢复机制更符合用户的真实需求。

本研究**不是**临床医学试验、心理治疗或医疗干预。

---

## 二、数据收集范围

在试用期间（7-14 天），我们将通过应用内的 `OVERLOAD_RESEARCH_MODE` 功能收集以下数据：

### 收集的数据

1. **匿名编码**（participantCode）：如 P01、P02，不关联您的真实身份
2. **会话标识**（sessionId）：每次签到的唯一编号
3. **会话日期**（date）：签到发生的日期
4. **是否首次使用**（firstUse）
5. **签到耗时**（checkInDurationMs）：完成签到所需的毫秒数
6. **是否查看负荷解释**（viewedLoadExplanation）
7. **是否打开降载选项**（openedDeLoad）
8. **建议决策**（suggestionDecision）：您选择了"接受"/"修改"/"拒绝"/"未提供建议"
9. **是否使用撤销**（usedUndo）
10. **次日是否返回**（returnedNextDay）
11. **是否感觉说教**（feltPreachy）
12. **是否有帮助**（foundHelpful）
13. **是否愿意继续使用**（willingToContinue）

### 数据收集方式

- 数据由应用自动记录，您无需手动填写（除每日简短反馈卡片外）
- 所有数据仅存储在您的设备本地
- 试用结束后，您通过应用内导出功能将数据发送给研究者
- 导出的数据经过掩码处理，不包含敏感内容

## 三、隐私保护措施

我们采取以下措施保护您的隐私：

1. **匿名化**：使用匿名编码（如 P01）标识参与者，不记录真实姓名、手机号等身份信息
2. **本地存储**：所有数据仅存储在您的设备本地，应用不申请网络权限，不会自动上传任何数据
3. **加密存储**：应用内的取件码等敏感字段使用 Android Keystore 加密存储
4. **掩码导出**：导出数据时，取件码、运单号等敏感字段自动掩码处理
5. **备份禁用**：应用已禁用系统备份和数据迁移，数据不会进入云端备份
6. **屏幕保护**：应用默认开启 FLAG_SECURE，防止截图和最近任务预览泄露信息
7. **生物认证**：查看完整取件码需要通过生物认证
8. **加密传输**：导出数据通过加密渠道传输给研究者

## 四、不收集的内容

以下内容我们**严格不收集**：

| 禁止收集项 | 说明 |
|------------|------|
| 真实姓名 | 不记录、不存储、不传输 |
| 真实签到文本 | 仅记录行为指标，不记录签到内容本身 |
| 真实身体状态描述 | 不记录任何健康或身体状态描述文本 |
| 设备序列号 | 不收集硬件标识符 |
| 手机号码 | 不收集个人联系方式 |
| 精确地理位置 | 不收集位置信息 |
| 通知原始正文 | 不保存第三方应用通知内容 |
| 照片或截图 | 不收集任何图像数据 |
| 通讯录或社交关系 | 不收集人际关系数据 |
| 生物特征数据 | 不收集指纹、面容等生物特征 |

## 十、风险与受益

### 可能的风险
- 时间投入：每日使用应用和回答反馈卡片约需 5-10 分钟
- 心理不适：部分用户可能在反思自身负荷状态时感到不适
- 隐私风险：尽管采取了多项保护措施，任何数据收集都存在残余隐私风险

### 可能的受益
- 更好地了解自身的负荷状态和调节习惯
- 为产品改进提供直接反馈，影响未来设计
- 试用期间免费使用产品全部功能
