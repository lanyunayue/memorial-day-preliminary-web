## 使用说明

- 每条研究必须包含完整字段，不得留空
- 不得编造研究来源、数据或结论
- 因果性字段必须明确区分"因果证据"和"相关性证据"
- 产品启示字段需说明该研究对 overload-recovery 产品的具体启示
- 所有引用须可追溯至原始来源

---

## 字段模板

| 字段 | 说明 |
|------|------|
| 编号 | PR-001 起递增 |
| 来源 | 论文标题 / 报告名称 |
| 机构 | 发表机构或期刊 |
| 时间 | 发表年份 |
| 样本量 | 研究样本量（如未报告标注 N/A） |
| 目标人群 | 研究涉及的人群特征 |
| 方法 | 研究方法（RCT / 队列 / 横断面 / 定性 / 元分析等） |
| 结论 | 研究核心结论（原文摘要级） |
| 局限 | 研究自述局限 + 我们识别的局限 |
| 因果性 | 因果证据 / 相关性证据 / 描述性 |
| 产品启示 | 对本产品的具体启示 |

## 引用规范

- 期刊论文：作者, 标题, 期刊名, 年份, 卷(期), 页码, DOI
- 机构报告：机构名, 报告标题, 年份, URL
- 元分析：标注纳入研究数量和总体效应量
- 预印本：标注 PREPRINT，不可作为 A 层最高可信度证据
