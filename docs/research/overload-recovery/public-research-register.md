## 主题一: 大学生过载与睡前拖延

### R-001: 睡前拖延与心理困扰的元分析

| 字段 | 内容 |
|------|------|
| **标题** | Bedtime procrastination and psychological distress in university students: a systematic review and meta-analysis of their association |
| **来源期刊** | Frontiers in Psychology |
| **发布机构** | Frontiers Media S.A. (石河子大学师范学院应用心理学系) |
| **时间** | 2026-03-05 |
| **样本量** | N = 35,097（18 项观察性研究） |
| **目标人群** | 大学生 |
| **研究方法** | 系统综述与元分析（随机效应模型），检索 PubMed、Web of Science、EBSCOhost、ScienceDirect、万方、ProQuest |
| **主要结论** | 睡前拖延与总体心理困扰显著正相关 (r = 0.34, 95% CI [0.23, 0.43])；与抑郁症状 (r = 0.28)、焦虑症状 (r = 0.30)、感知压力 (r = 0.38) 均显著正相关。压力关联最强，支持"睡前拖延是压力驱动的自我调节失败"理论模型。亚洲样本的关联更强。 |
| **局限** | 全部为横断面研究，无法推断因果方向；异质性较高 (I² > 90%)；未区分主动与被动睡前拖延。 |
| **因果性** | 观察性关联，非因果证据 |
| **产品启示** | 睡前拖延与压力的关联 (r = 0.38) 为"深夜隐藏任务压力降低焦虑"假设 (H-002) 提供部分支持。大学生是核心目标人群，该群体睡前拖延与压力的关联已被大样本验证。 |
| **DOI** | 10.3389/fpsyg.2026.1767938 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/41867997/ |

### R-002: 大学生学业压力画像与睡前拖延

| 字段 | 内容 |
|------|------|
| **标题** | Academic stress profiles and bedtime procrastination among college students: the mediating role of rumination and the moderating effect of perceived stress |
| **来源期刊** | Frontiers in Psychology |
| **发布机构** | Frontiers Media S.A. (温州大学、河南工业大学、汕头大学) |
| **时间** | 2026-04-22 |
| **样本量** | N = 741（中国大学生） |
| **目标人群** | 中国大学生 |
| **研究方法** | 横断面研究，潜在剖面分析 (LPA) + 调节中介模型 |
| **主要结论** | 识别出四类学业压力画像：压力不敏感型 (11%)、适应良好型 (32%)、普遍负担型 (45%)、不堪重负型 (12%)。不堪重负组睡前拖延得分最高。症状反刍是不适应中介 (β = 0.24, p = 0.002)，反思是适应中介 (β = -0.22, p < 0.01)。感知压力在高水平时才激活不适应路径。 |
| **局限** | 横断面设计无法确定因果方向；自我报告数据可能存在偏差；仅中国样本，文化普适性待验证。 |
| **因果性** | 横断面关联，非因果证据 |
| **产品启示** | 不同压力画像的大学生需要差异化的干预策略。"不堪重负型"用户最需要降载支持，而"普遍负担型"在高感知压力下也需要干预。支持个性化负荷管理的价值。 |
| **DOI** | 10.3389/fpsyg.2026.1784775 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/42099722/ |

## 主题二: 工作边界与恢复

### R-003: 工作心理脱离预测心理健康的前瞻性纵向研究

| 字段 | 内容 |
|------|------|
| **标题** | Psychological detachment from work predicts mental wellbeing of working-age adults: Findings from the 'Wellbeing of the Workforce' (WoW) prospective longitudinal cohort study |
| **来源期刊** | PLOS ONE |
| **发布机构** | Public Library of Science (诺丁汉大学、女王大学贝尔法斯特、莱斯特大学) |
| **时间** | 2025-01-14 |
| **样本量** | T1 n = 337, T2 n = 169（两波前瞻性纵向队列） |
| **目标人群** | 英国在职成年人（雇佣或自雇） |
| **研究方法** | 前瞻性纵向队列研究，两波数据采集 (2020年4-6月和2021年3-4月)，描述性统计与参数/非参数推断 |
| **主要结论** | T1 时低心理脱离患病率为 21.4%，T2 降至 16.0%。T1 时更好的心理脱离预测 T2 时更好的心理健康 (β = .21, p = .01)、更低焦虑 (β = -.43, p < .001) 和更高生活满意度 (β = .32, p = .003)。心理脱离的改善也独立预测更好的结果。 |
| **局限** | 样本流失 (337→169)；COVID-19 背景可能影响普适性；自报告数据；未测量具体恢复活动类型。 |
| **因果性** | 前瞻性纵向设计提供时序性证据，但非随机对照实验 |
| **产品启示** | 心理脱离（从工作中"断开"）是恢复的关键机制。支持 H-003（Waiting For 会形成持续负荷），因为未完成的工作承诺会阻碍心理脱离。产品应帮助用户"关闭"工作认知，而非仅管理任务列表。 |
| **DOI** | 10.1371/journal.pone.0312673 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/39808595/ |

| 字段 | 内容 |
|------|------|
| **标题** | Understanding and overcoming barriers to digital health adoption: a patient and public involvement study |
| **来源期刊** | Translational Behavioral Medicine |
| **发布机构** | Oxford University Press / Society of Behavioral Medicine (新加坡-ETH中心、新加坡国立大学、ETH Zurich) |
| **时间** | 2025-01-16 |
| **样本量** | N = 118（多文化社区参与者） |
| **目标人群** | 新加坡多元文化社区成年人 |
| **研究方法** | 患者与公众参与 (PPI) 方法，训练社区引导员在公共餐饮场所进行半结构化讨论，归纳主题分析并映射到行为改变理论 |
| **主要结论** | 识别出 5 个关键障碍主题：(a) 对数字健康方案的认知不足，(b) 权衡收益与负担，(c) 可及性，(d) 对开发者与技术的信任，(e) 用户体验影响。映射到 13 个行为机制和 26 个行为改变技术。提出 5 项策略：社区推广与数字素养培训、医疗场景简短咨询、与个人价值挂钩的可变奖励、可及性政策保障、游戏化用户友好设计。 |
| **局限** | 单一城市 (新加坡) 样本；PPI 方法非传统研究设计；未追踪后续行为改变；参与者可能存在选择偏差。 |
| **因果性** | 探索性定性研究，无因果推断 |
| **产品启示** | "权衡收益与负担"是采用障碍的核心主题。支持 H-004（十秒签到的留存高于长篇日记），因为数据输入负担是采用障碍的关键因素。产品必须在首次接触时清晰传达价值，并最小化使用负担。 |
| **DOI** | 10.1093/tbm/ibaf010 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/40167046/ |

## 主题四: 睡眠应用评估

### R-005: 睡眠自我管理应用系统评估

| 字段 | 内容 |
|------|------|
| **标题** | Smartphone Applications to Support Sleep Self-Management: Review and Evaluation |
| **来源期刊** | Journal of Clinical Sleep Medicine (J Clin Sleep Med) |
| **发布机构** | American Academy of Sleep Medicine (华盛顿大学护理学院、宾夕法尼亚大学护理学院) |
| **时间** | 2018-10-15 |
| **样本量** | 73 个应用（从 2,431 个候选中筛选） |
| **目标人群** | 消费级睡眠自我管理应用用户 |
| **研究方法** | 系统评估，使用移动应用评级量表 (MARS) 和 IMS 功能评分对四个应用商店 (iTunes、Google Play、Amazon、Microsoft) 中的应用进行评价 |
| **主要结论** | MARS 中位得分 3.1/5.0，仅 58% (42/73) 达到最低可接受分。所有应用都有记录功能，但仅 8 个有干预功能。45% (33/73) 使用传感器自动采集数据，27 个允许手动输入，14 个支持两种方式。缺乏验证研究是限制临床价值的主要问题。 |
| **局限** | 评估时点为 2017-2018 年，应用更新频繁；仅评估功能和质量，未测试临床有效性；MARS 评分主观性。 |
| **因果性** | 描述性评估，无因果推断 |
| **产品启示** | 手动输入睡眠数据在睡眠应用中已被广泛接受 (27/73 个应用支持)，支持 H-005（用户愿意记录大概睡眠时间）。但大多数应用缺乏干预功能，仅停留在数据采集层面。"负荷与恢复助手"应超越记录，提供可操作的恢复建议。 |
| **DOI** | 10.5664/jcsm.7396 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/30353814/ |

### R-006: 睡眠应用的当前局限与挑战

| 字段 | 内容 |
|------|------|
| **标题** | Sleep apps: current limitations and challenges |
| **来源期刊** | Sleep Science (Sleep Sci) |
| **发布机构** | Associação Fundadora de Pesquisa e Desenvolvimento Científico (West Hertfordshire Hospitals NHS Trust) |
| **时间** | 2021-01 至 03 |
| **样本量** | 综述（无原始数据采集） |
| **目标人群** | 睡眠应用用户与临床医生 |
| **研究方法** | 叙事综述 |
| **主要结论** | 睡眠应用存在三大问题：(1) 缺乏经验证据支持其声称，且已有证据常存在显著方法学局限；(2) 数据隐私问题严重，应用在用户不知情的情况下与商业和营销合作伙伴共享敏感数据；(3) 可能增加医疗系统负担——部分应用产生大量假阳性，临床医生需更多时间分析数据。未来需严格验证研究并赋予用户数据共享自主权。 |
| **局限** | 叙事综述无系统检索策略；作者单一；可能存在选择偏差。 |
| **因果性** | 观点性综述，无因果推断 |
| **产品启示** | 隐私问题是睡眠应用的关键批评。本地优先架构（数据不上传）可作为核心差异化优势。避免过度承诺（如"精准睡眠监测"），诚实说明功能局限。不要增加用户焦虑（如"你的睡眠质量很差"），而应提供可操作的改善建议。 |
| **DOI** | 10.5935/1984-0063.20200036 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/34104344/ |

## 主题五: 健康应用弃用原因

### R-007: 美国手机用户健康应用使用全国调查

| 字段 | 内容 |
|------|------|
| **标题** | Health App Use Among US Mobile Phone Owners: A National Survey |
| **来源期刊** | JMIR mHealth and uHealth |
| **发布机构** | JMIR Publications (纽约大学医学院人口健康系) |
| **时间** | 2015-11-04 |
| **样本量** | N = 1,604（美国全国手机用户） |
| **目标人群** | 美国手机用户 |
| **研究方法** | 横断面全国调查，36 项问卷评估社会人口学特征、健康应用使用/不使用原因、停止使用原因及一般健康状况 |
| **主要结论** | 58.23% (934/1604) 下载过健康应用。在下载者中，45.7% (427/934) 停止使用部分应用。停止使用的首要原因：高数据输入负担 (high data entry burden)、失去兴趣、隐藏费用。成本是重大顾虑，大部分用户不愿为健康应用付费。但已下载者对准确性和数据安全的信任度较高。 |
| **局限** | 横断面设计无法追踪使用变化；自我报告可能存在回忆偏差；2015 年数据可能不反映当前市场。 |
| **因果性** | 横断面关联，非因果证据 |
| **产品启示** | 数据输入负担是弃用的首要原因，强烈支持 H-004（十秒签到的留存高于长篇日记）。隐藏费用导致弃用——产品应避免自动续费陷阱。成本敏感度高——免费或一次性付费模式更友好。 |
| **DOI** | 10.2196/mhealth.4924 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/26537656/ |

### R-008: 身体活动应用采用与持续使用的纵向定性研究

| 字段 | 内容 |
|------|------|
| **标题** | Exploring Users' Experiences of the Uptake and Adoption of Physical Activity Apps: Longitudinal Qualitative Study |
| **来源期刊** | JMIR mHealth and uHealth |
| **发布机构** | JMIR Publications (米兰比可卡大学心理学系、伦敦大学学院) |
| **时间** | 2019-02-08 |
| **样本量** | 基线 N = 20，2 周随访 n = 17 |
| **目标人群** | 潜在身体活动应用用户 |
| **研究方法** | 纵向单臂定性研究，基线"出声思考"探索 + 2 周后半结构化访谈，归纳主题分析 |
| **主要结论** | 促进公平简洁的用户体验、支持自我调节技能、满足运动动机的功能在首次接触和 2 周使用后均被认为重要。支持关联需求和促进意图实施的功能主要在首次接触时重要。主动和个性化功能（整合行为、心理和情境信息以提供适应性计划和即时支持）对持续参与至关重要。 |
| **局限** | 样本量小 (n=17)；单一国家 (意大利)；2 周随访期短；仅身体活动应用，推广性有限。 |
| **因果性** | 定性探索，无因果推断 |
| **产品启示** | 主动和个性化功能对持续使用至关重要。支持"负荷与恢复助手"应提供个性化建议而非仅记录。首次接触体验决定用户是否继续使用——onboarding 必须简洁且清晰传达价值。从 20 人到 17 人的流失 (15%) 说明短期弃用率高，需特别关注首周留存。 |
| **DOI** | 10.2196/11636 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/30735143/ |

## 主题六: 冥想应用评估

| 字段 | 内容 |
|------|------|
| **标题** | Review and Evaluation of Mindfulness-Based iPhone Apps |
| **来源期刊** | JMIR mHealth and uHealth |
| **发布机构** | JMIR Publications (昆士兰科技大学健康与生物医学创新研究所) |
| **时间** | 2015-08-19 |
| **样本量** | 23 个应用（从 700 个候选中筛选） |
| **目标人群** | 正念/冥想应用用户 |
| **研究方法** | 系统评估，使用移动应用评级量表 (MARS) 的参与度、功能性、视觉美学、信息质量和主观质量子量表进行专家评分，双人评分评估评分者间信度 |
| **主要结论** | MARS 中位得分 3.2/5.0，刚超过最低可接受分 3.0。Headspace 得分最高 (4.0)，其次为 Smiling Mind (3.7)、iMindfulness (3.5)、Mindfulness Daily (3.5)。大多数声称正念相关的应用实际只是引导冥想、计时器或提醒器，缺乏高质量正念训练。正念应用的有效性证据极少。 |

| **因果性** | 描述性评估，无因果推断 |
| **产品启示** | 即使是得分最高的冥想应用 (Headspace 4.0/5.0) 在参与度和信息质量上仍有提升空间。大多数应用停留在"工具"层面（计时器、引导音频），缺乏个性化训练。"负荷与恢复助手"的恢复模块应超越简单工具，提供基于负荷状态的个性化恢复建议。 |
| **DOI** | 10.2196/mhealth.4328 |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/26290327/ |

## 主题七: 非临床安全边界

### R-010: 抑郁与自杀预防应用的临床指南依从性评估

| 字段 | 内容 |
|------|------|
| **标题** | Suicide prevention and depression apps' suicide risk assessment and management: a systematic assessment of adherence to clinical guidelines |
| **来源期刊** | BMC Medicine |
| **发布机构** | BioMed Central (南洋理工大学李光前医学院、阿姆斯特丹大学医学中心、昆士兰科技大学) |
| **时间** | 2019-12-19 |
| **样本量** | 69 个应用（从 2,690 个候选中筛选） |
| **目标人群** | 抑郁与自杀预防应用用户 |
| **研究方法** | 系统评估，使用 50 题检查表评估 6 项循证自杀预防策略的依从性（情绪与自杀意念追踪、安全计划制定、推荐活动、信息教育、支持网络接入、紧急咨询接入），并基于 HONcode 原则评估可信度 |
| **主要结论** | 仅 5/69 (7%) 应用纳入了全部 6 项自杀预防策略。6 个应用 (6/69, 9%) 提供了错误的危机热线号码，其中 2 个应用在两个应用商店均可下载且下载量均超百万次。大多数应用包含紧急联系信息 (65/69, 94%) 和直接危机热线接入 (46/69, 67%)。应用商店和应用行业的自我治理、质量和安全保证存在失败。 |
| **局限** | 评估时点为 2019 年 1 月；检查表为研究团队开发，未经过正式验证；仅评估内容存在性，未评估质量或有效性。 |
| **因果性** | 描述性评估，无因果推断 |
| **产品启示** | 心理健康应用领域存在严重的安全治理缺失。"负荷与恢复助手"必须明确界定非临床定位——不提供自杀风险评估或精神疾病诊断。需要建立安全边界：明确声明"本应用不是医疗设备"，在检测到高风险信号时引导用户寻求专业帮助。不提供可能有害的建议。 |
| **DOI** | 10.1186/s12916-019-1461-z |
| **PubMed URL** | https://pubmed.ncbi.nlm.nih.gov/31852455/ |

## 使用规则

1. 每条记录的 DOI 和 PubMed URL 均已通过 PubMed 数据库验证，可直接访问。
2. 因果性等级从高到低：随机对照实验 > 前瞻性纵向 > 横断面 > 定性/描述性 > 观点性。
3. "产品启示"字段记录该研究对"负荷与恢复助手"产品的直接启示，可引用于产品决策文档。
4. 引用时请注明证据编号 (如 R-001)、来源期刊和 DOI。
5. 所有研究的局限应在引用时一并说明，避免过度解读。
