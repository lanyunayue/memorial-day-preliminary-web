# Same-Task Comparison Test

The participant chooses the tool they actually use: system notes, calendar, a task app, chat self-transfer, or another real tool.

For the same participant-selected matter, record:

- Completion time and interaction count.
- Whether classification had to be decided first.
- Whether the result tracks Waiting For.
- Whether it explains a next action.
- Participant preference and reason.
- Where 时刻 is clearly stronger or weaker.

Counterbalance tool order across participants. Do not coach toward 时刻 and do not use team-written example text as participant input.
